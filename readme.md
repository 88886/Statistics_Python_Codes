# 《统计学原理实验教程（Python）》书中代码实现

## 简介

《统计学原理实验教程（Python）》是厦门大学出版社在2019年出版的图书，主要通过python来实现统计学中最基本的检验方法。个人觉得本书内容较为基础，适合初学者进行学习。结构体系化，由浅入深。是一本很好的教科书。

上网搜寻随书代码无果，遂决定自行敲代码。
